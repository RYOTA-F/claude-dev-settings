# ブランドガイド（Brand Guide）

## 概要

プロダクトのブランドアイデンティティを定義・維持し、一貫したコミュニケーションを実現するスキル。

## 責務

### ブランドアイデンティティ

- ブランドのミッション・ビジョン・バリューを明文化する
- ブランドパーソナリティ（どんな人格として認知されたいか）を定義する
- ビジュアルアイデンティティ（ロゴ、カラー、タイポグラフィ）のガイドラインを管理する

### トーン＆ボイス

- ブランドの声のトーンを定義する:
  - フォーマル度（カジュアル〜フォーマル）
  - 感情（冷静〜情熱的）
  - 専門度（初心者向け〜専門家向け）
- チャネル別のトーン調整ガイドラインを提供する
- 使用する言葉・避ける言葉のリストを管理する

### メッセージング

- プロダクトの価値提案を簡潔に伝えるメッセージを策定する
- ターゲットセグメント別のメッセージバリエーションを用意する
- タグライン・キャッチコピーの作成・評価を行う

## 出力フォーマット

```text
## ブランドガイドライン
### ブランドパーソナリティ

- キーワード: [3〜5 語]
- トーン: [フォーマル度 / 感情 / 専門度]

### メッセージング

- タグライン: ...
- エレベーターピッチ（30 秒）: ...
- 価値提案: ...

### Do / Don't
| Do | Don't |
| ---- | ------- |
| ... | ... |
```text
