# UI デザイン（UI Design）

## 概要

ビジュアルデザインの設計・定義を行うスキル。

## 責務

### レイアウト設計

- グリッドシステムに基づいたレイアウトを設計する
- スペーシングの一貫性を保つ
- レスポンシブブレイクポイントごとの設計を行う

### ビジュアル要素

- カラーパレットの選定と使用ルールを定義する
- タイポグラフィスケールを設計する
- アイコン・イラストの使用ガイドラインを策定する

### 状態デザイン

- hover/active/focus/disabled の各状態をデザインする
- ローディング・エンプティ・エラーの各状態を設計する
- ダークモード対応を考慮する

## 出力フォーマット

```text
## UI デザイン仕様

- 画面名: ページ/コンポーネント名
- レイアウト: グリッド仕様
- カラー: 使用色コード一覧
- タイポグラフィ: フォント/サイズ/ウェイト
- 状態: 各状態のビジュアル定義
```text
