# リソース配分（Resource Allocation）

## 概要

人的リソース・ツール・外注のコスト効率を分析し、最適な配分を提案するスキル。

## 責務

### 人的リソースのコスト分析

- チームメンバーの稼働状況と工数を可視化する
- タスク種別ごとの工数配分を分析する
- 過負荷・低稼働のリソースを特定する

### 外注判断

- 内製 vs 外注のコスト比較を行う
- 外注に適した領域を特定する（コアではない、一時的な需要、専門知識が必要等）
- 外注のリスク（品質、コミュニケーション、知識流出）を評価する

### ツール投資判断

- 開発ツール・SaaS の費用対効果を評価する
- 無料ツールと有料ツールの比較を行う
- ツール導入による生産性向上を定量化する

## 出力フォーマット

```

|------|----------|----------|------|------|

### ツール投資

```
