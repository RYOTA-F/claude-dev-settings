# CFO エージェント（Plutus）スキル一覧

Plutus の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせて財務管理を行う。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| コスト最適化 | [cost-optimization.md](cost-optimization.md) | インフラコスト分析、無駄の削減、料金プラン最適化 |
| ROI 分析 | [roi-analysis.md](roi-analysis.md) | 投資対効果の算出、施策の費用便益分析 |
| 予算計画 | [budget-planning.md](budget-planning.md) | 予算策定、配分計画、執行管理 |
| 財務レポート | [financial-report.md](financial-report.md) | 定期財務報告、コストトレンド分析、予実管理 |
| リソース配分 | [resource-allocation.md](resource-allocation.md) | 人的リソースのコスト分析、外注判断、ツール投資判断 |
