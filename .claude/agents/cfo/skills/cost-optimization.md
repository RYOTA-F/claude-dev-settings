# コスト最適化（Cost Optimization）

## 概要

インフラ・ツール・サービスのコストを分析し、無駄を削減しながら必要な投資を維持するスキル。

## 責務

### インフラコスト分析

- AWS の各サービス利用料を月次で可視化する
- 使用率の低いリソース（未使用 EC2、過剰プロビジョニング等）を特定する
- リザーブドインスタンス、Savings Plans の適用余地を評価する

### 無駄の削減

- 不要なリソース・サービスの棚卸しを定期的に実施する
- 開発環境・ステージング環境のコスト効率を検証する
- ログ・ストレージの保持期間を最適化する

### 料金プラン最適化

- SaaS ツールのプラン・ライセンス数を適正化する
- 年間契約とオンデマンドのコスト比較を行う
- ボリュームディスカウントの交渉余地を評価する

## 出力フォーマット

```
## コスト最適化レポート
### 月次コストサマリー
- 総額: ¥XXX,XXX
- 前月比: +X% / -X%

### 削減機会
| 項目 | 現在のコスト | 最適化後 | 削減額 | 優先度 |
|------|------------|---------|--------|--------|

### 推奨アクション
1. [即時実行] ...
2. [計画的に] ...
```
