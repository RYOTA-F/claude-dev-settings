# 予算計画（Budget Planning）

## 概要

プロジェクトの予算策定・配分・執行管理を行うスキル。

## 責務

### 予算策定

- 四半期・年次の予算枠を設定する
- カテゴリ別（インフラ、ツール、外注、教育等）に配分する
- 予備費（バッファ）を適切に確保する（全体の 10〜15%）

### 配分計画

- 優先度に基づいてカテゴリ間の配分比率を決める
- 固定費と変動費を区別する
- 新規投資と維持費の比率を明確にする

### 執行管理

- 月次で予算消化率をトラッキングする
- 予算超過の兆候を早期に検知する
- 予算の再配分が必要な場合に提案する

## 出力フォーマット

```text
## 予算ステータス（YYYY/MM）
### 全体

- 予算: ¥XXX / 消化: ¥XXX (XX%)
- 残: ¥XXX

### カテゴリ別
| カテゴリ | 予算 | 消化 | 消化率 | ステータス |
| --------- | ------ | ------ | -------- | ----------- |
| インフラ | | | | 正常/注意/超過 |
| ツール | | | | |

### アラート

- [カテゴリ] が予算の XX% に到達。対策: ...
```text
