# 財務レポート（Financial Report）

## 概要

プロジェクトの財務状況を定期的に報告し、コストトレンドと予実の乖離を可視化するスキル。

## 責務

### 定期財務報告

- 月次で収支サマリーを作成する
- 主要コストドライバーを特定する
- 前月・前年同月との比較を提供する

### コストトレンド分析

- コストの推移をカテゴリ別に分析する
- 急激な変動の原因を特定する
- 将来のコスト予測を提供する

### 予実管理

- 予算と実績の乖離を可視化する
- 乖離が大きい項目の原因を分析する
- 着地見込みを算出し、必要に応じて是正策を提案する

## 出力フォーマット

```
## 月次財務レポート（YYYY/MM）
### サマリー
- 総コスト: ¥XXX（予算比 XX%）
- 前月比: +¥XX (+X%)
- 着地見込み: ¥XXX（年間予算比 XX%）

### トレンド
| 月 | コスト | 前月比 |
|----|--------|--------|

### 予実乖離
| カテゴリ | 予算 | 実績 | 乖離 | 原因 |
|---------|------|------|------|------|

### 要対応事項
1. ...
```
