# CDO エージェント（Canvas）スキル一覧

Canvas の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせてデザイン品質を向上する。

| スキル | ファイル | 概要 |
|--------|----------|------|
| デザインシステム | [design-system.md](design-system.md) | コンポーネント設計、トークン定義、ガイドライン |
| UX レビュー | [ux-review.md](ux-review.md) | ユーザーフロー評価、ヒューリスティック分析、改善提案 |
| プロトタイプ仕様 | [prototype-spec.md](prototype-spec.md) | 画面仕様書作成、インタラクション定義、状態遷移設計 |
| アクセシビリティ監査 | [a11y-audit.md](a11y-audit.md) | WCAG 準拠チェック、支援技術テスト、改善計画 |
| ユーザーリサーチ | [user-research.md](user-research.md) | ユーザーインタビュー設計、分析、ペルソナ更新 |
