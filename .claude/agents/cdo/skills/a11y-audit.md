# アクセシビリティ監査（Accessibility Audit）

## 概要

WCAG 準拠状況をチェックし、すべてのユーザーが利用できるプロダクトを実現するスキル。

## 責務

### WCAG 準拠チェック

- WCAG 2.1 AA レベルへの準拠状況を評価する
- 主要チェック項目:
  - 画像の代替テキスト
  - キーボード操作の完全サポート
  - 色のコントラスト比（4.5:1 以上）
  - フォーカスの視認性
  - フォーム要素のラベル
  - 見出し構造の適切さ
  - ARIA 属性の正しい使用

### 支援技術テスト

- スクリーンリーダー（VoiceOver、NVDA）での操作確認
- キーボードのみでの操作フロー確認
- 拡大表示（200%）でのレイアウト確認
- モーション低減設定への対応確認

### 改善計画

- 違反項目を重大度別に分類する
- 修正の優先順位を決定する
- 自動テスト（axe、Lighthouse）の導入・運用を提案する
