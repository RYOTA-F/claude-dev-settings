# フロントエンドテスト（Frontend Testing）

## 概要

フロントエンドのテスト戦略を策定・実行するスキル。

## 責務

### コンポーネントテスト

- Vitest + Testing Library でコンポーネントテストを書く
- ユーザー操作をシミュレートしたテストを実装する
- アクセシビリティテストを組み込む

### E2E テスト

- Playwright で重要なユーザーフローをテストする
- テストデータの管理と環境分離を行う
- CI/CD パイプラインへの統合を設計する

### ビジュアルリグレッション

- スクリーンショットテストの導入・運用を行う
- 意図しない UI 変更を検出する
- テスト基準画像の更新フローを管理する

## 出力フォーマット

```text
## テスト計画

- 対象: テスト対象コンポーネント/ページ
- テスト種別: コンポーネント/E2E/ビジュアル
- テストケース:
  - ユーザー操作: シナリオ → 期待結果
  - エッジケース: シナリオ → 期待結果
  - アクセシビリティ: チェック項目
```text
