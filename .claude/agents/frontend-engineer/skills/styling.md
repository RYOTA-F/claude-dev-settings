# スタイリング（Styling）

## 概要

CSS 設計・レスポンシブ対応・アニメーション実装を行うスキル。

## 責務

### CSS 設計

- プロジェクトの CSS 方針に従ったスタイル実装を行う
- デザイントークン（色、サイズ、スペーシング）を統一管理する
- ダークモード対応を考慮する

### レスポンシブ対応

- モバイルファーストで設計する
- ブレイクポイントの標準を定義・適用する
- タッチ操作とマウス操作の両方に対応する

### アニメーション

- パフォーマンスに配慮したアニメーションを実装する
- `prefers-reduced-motion` を考慮する
- トランジションの一貫性を保つ

## 出力フォーマット

```text
## スタイリング方針

- 対象コンポーネント: コンポーネント名
- ブレイクポイント対応: モバイル/タブレット/デスクトップ
- デザイントークン: 使用するトークン一覧
- アニメーション: トランジション仕様
```text
