# パフォーマンス最適化（Frontend Performance）

## 概要

フロントエンドのパフォーマンスを計測・最適化するスキル。

## 責務

### Core Web Vitals

- LCP、FID、CLS を計測・改善する
- パフォーマンスバジェットを設定する
- 継続的なモニタリングを導入する

### バンドル最適化

- コード分割とダイナミックインポートを活用する
- Tree-shaking の効果を最大化する
- サードパーティライブラリの影響を分析する

### レンダリング最適化

- 不要な再レンダリングを検出・排除する
- メモ化の適切な適用判断を行う
- SSR/SSG/ISR の使い分けを設計する

## 出力フォーマット

```text
## パフォーマンス改善提案

- 対象: ページ/コンポーネント
- 現状指標: LCP/FID/CLS の計測値
- 改善施策: 具体的な対策
- 期待効果: 改善後の目標値
```text
