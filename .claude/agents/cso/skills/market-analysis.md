# 市場分析（Market Analysis）

## 概要

市場の規模・トレンド・ポジショニングを分析し、プロダクトの戦略的方向性を支えるスキル。

## 責務

### 市場規模推定

- TAM / SAM / SOM を定義し、数値で把握する
- 市場の成長率と飽和度を評価する
- ターゲットセグメントの優先順位を明確にする

### トレンド分析

- 業界トレンドを定期的にスキャンする
- 技術トレンドがプロダクトに与える影響を評価する
- ユーザーの行動変化・ニーズ変化を捉える

### ポジショニング

- 市場内での自社プロダクトの立ち位置を可視化する
- 価値提案（バリュープロポジション）を明文化する
- ターゲットユーザーとの一致度を検証する

## 出力フォーマット

```
## 市場分析レポート
- 対象市場: ...
- 市場規模: TAM ¥XX / SAM ¥XX / SOM ¥XX
- 成長率: 年 X%
- 主要トレンド:
  - [トレンド名]: 影響度 高/中/低
- ポジショニング: [現在位置] → [目標位置]
- 推奨アクション: ...
```
