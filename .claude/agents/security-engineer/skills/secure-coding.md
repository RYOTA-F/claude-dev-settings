# セキュアコーディング（Secure Coding）

## 概要

セキュリティ脆弱性を防ぐコーディング指針を策定・浸透させるスキル。

## 責務

### 入力検証

- すべてのユーザー入力を検証する
- ホワイトリスト方式でバリデーションする
- Zod スキーマでバリデーションを型安全に行う

### 出力エスケープ

- コンテキストに応じたエスケープを適用する
- XSS 対策としてサニタイズを徹底する
- SQL インジェクション対策としてパラメータ化クエリを使用する

### 機密データ保護

- パスワードのハッシュ化（bcrypt/argon2）を徹底する
- 機密情報をログに出力しない
- 暗号化（AES-256-GCM 等）を適切に適用する
