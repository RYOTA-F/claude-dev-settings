# セキュリティエンジニアエージェント（Ares）スキル一覧

Ares の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせてプロダクトのセキュリティを強化する。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| 脆弱性診断 | [vulnerability-assessment.md](vulnerability-assessment.md) | OWASP Top 10、コードレビュー、依存関係スキャン |
| 認証・認可設計 | [auth-design.md](auth-design.md) | 認証フロー設計、RBAC/ABAC、トークン管理 |
| セキュアコーディング | [secure-coding.md](secure-coding.md) | インジェクション対策、入力検証、出力エスケープ |
| インシデント対応 | [incident-response.md](incident-response.md) | 対応計画、トリアージ、フォレンジック、事後分析 |
| コンプライアンス | [compliance.md](compliance.md) | セキュリティポリシー、監査対応、規制準拠 |
