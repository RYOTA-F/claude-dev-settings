# スケーラビリティ評価（Scalability Assessment）

## 概要

システムの性能を評価し、ボトルネックの特定とキャパシティプランニングを行うスキル。

## 責務

### 性能評価

- API レスポンスタイム、スループットを計測する
- データベースクエリの実行時間を分析する
- フロントエンドのレンダリングパフォーマンスを評価する

### ボトルネック特定

- 負荷が集中するコンポーネントを特定する
- N+1 クエリ、不要な再計算、メモリリークを検知する
- ネットワーク・I/O の遅延要因を分析する

### キャパシティプランニング

- 現在の利用状況から将来の負荷を予測する
- スケールアウト / スケールアップの判断基準を定義する
- コストとパフォーマンスのバランスを最適化する
