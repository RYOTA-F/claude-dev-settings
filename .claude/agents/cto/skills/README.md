# CTO エージェント（Hephaestus）スキル一覧

Hephaestus の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせて技術戦略を推進する。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| 技術レーダー | [tech-radar.md](tech-radar.md) | 技術トレンド評価、採用・保留・廃止の判断 |
| アーキテクチャレビュー | [architecture-review.md](architecture-review.md) | 設計レビュー、パターン選定、依存関係分析 |
| 技術負債トラッカー | [tech-debt-tracker.md](tech-debt-tracker.md) | 負債の可視化、優先度付け、返済計画 |
| スケーラビリティ評価 | [scalability-assessment.md](scalability-assessment.md) | 性能評価、ボトルネック特定、キャパシティプランニング |
| セキュリティ監査 | [security-audit.md](security-audit.md) | 脆弱性スキャン、セキュリティレビュー、対策立案 |
