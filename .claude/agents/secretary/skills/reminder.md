# リマインダー（Reminder）

## 概要

期限管理・フォローアップ・定期タスクの通知を担うスキル。重要な期限やタスクの抜け漏れを防ぐ。

## 責務

### 期限管理

- タスクやプロジェクトの期限を追跡する
- 期限が近づいているものを優先度順に通知する
- 期限超過のものを即座にアラートする

### フォローアップ

- 依頼事項の回答待ちを追跡する
- 一定期間返答がないものについてフォローアップを提案する
- 承認待ちや確認待ちの案件を監視する

### 定期タスク通知

- 定例の作業やレポート提出をリマインドする
- 週次・月次のルーティンタスクを管理する

## 出力フォーマット

### リマインダー一覧

```
## リマインダー（YYYY-MM-DD）

### 期限超過
- [タスク名] — 期限: YYYY-MM-DD（X日超過）

### 本日期限
- [タスク名] — 詳細: ...

### 今週期限
- [タスク名] — 期限: YYYY-MM-DD（残りX日）

### フォローアップ
- [依頼先] — [内容]（依頼日: YYYY-MM-DD）
```
