# 秘書エージェント（Iris）スキル一覧

Iris の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせて秘書業務を遂行する。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| スケジュール管理 | [schedule.md](schedule.md) | カレンダー管理、日程調整、リマインド |
| 情報整理 | [information.md](information.md) | 議事録作成、要約、ナレッジ整理 |
| メール・連絡管理 | [communication.md](communication.md) | メール処理、下書き作成、連絡のトリアージ |
| リマインダー | [reminder.md](reminder.md) | 期限管理、フォローアップ、定期タスク通知 |
