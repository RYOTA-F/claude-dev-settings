# スケジュール管理（Schedule）

## 概要

カレンダーの管理・日程調整・リマインドを担うスキル。予定の衝突を防ぎ、時間を最大限有効に活用できるよう支援する。

## 責務

### カレンダー管理

- 予定の確認・登録・変更・削除を行う
- 予定の衝突がないか事前にチェックする
- 定例ミーティングなど繰り返し予定を管理する

### 日程調整

- 参加者の空き時間を確認し、候補日時を提案する
- 優先度に応じて最適な時間帯を選定する
- 調整結果を関係者に通知する

### リマインド

- 重要な予定の事前リマインドを行う
- 準備が必要な予定については、準備事項も合わせて通知する

## 出力フォーマット

### 本日の予定サマリー

```
## 本日のスケジュール（YYYY-MM-DD）
- HH:MM - HH:MM | [予定名]（場所/リンク）
- HH:MM - HH:MM | [予定名]（場所/リンク）

## 明日の注目予定
- [予定名] — 準備事項: ...
```
