# 事業計画策定（Business Plan）

## 概要

中期経営計画・年度計画の策定と進捗管理を行うスキル。目標から逆算した実行計画を設計する。

## 責務

### 中期経営計画

- 3〜5 年のビジョンと定量目標を設定する
- 事業ドメインの選択と集中を明確にする
- フェーズごとのマイルストーンを定義する

### 年度計画

- 中期計画を年度単位に分解する
- 四半期ごとの具体的なアクションプランを策定する
- 部門別の目標と責任を割り当てる

### 進捗モニタリング

- 計画と実績の差異を定量的に把握する
- 遅延・乖離の原因を分析し、是正策を提案する
- 計画の見直しタイミングと基準を定義する

## 出力フォーマット

```text
## 事業計画書

### ビジョン

- [期間]: [目指す姿]

### 定量目標
| 指標 | 現状 | 目標 | 達成期限 |
| ---- | ---- | ---- | -------- |

### マイルストーン
| フェーズ | 期間 | 主要施策 | 成果指標 |
| -------- | ---- | -------- | -------- |

### リスクと対策

- [リスク]: 影響度 高/中/低 → 対策: ...
```text
