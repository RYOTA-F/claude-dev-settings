# 経営企画エージェント（Prometheus）スキル一覧

Prometheus の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせて経営企画業務を行う。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| 事業計画策定 | [business-plan.md](business-plan.md) | 中期経営計画、年度計画、マイルストーン設計 |
| 経営会議支援 | [executive-meeting.md](executive-meeting.md) | アジェンダ設計、論点整理、意思決定フレームワーク |
| 部門間調整 | [cross-functional.md](cross-functional.md) | 部門横断プロジェクト推進、リソース調整、合意形成 |
| 新規事業評価 | [new-business.md](new-business.md) | 事業機会評価、フィージビリティスタディ、Go/No-Go 判定 |
| KPI ダッシュボード | [kpi-dashboard.md](kpi-dashboard.md) | 経営指標設計、定点観測、アラート設計 |
