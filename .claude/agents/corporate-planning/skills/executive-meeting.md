# 経営会議支援（Executive Meeting）

## 概要

経営会議の生産性を最大化するため、アジェンダの設計・論点整理・意思決定の構造化を行うスキル。

## 責務

### アジェンダ設計

- 議題の優先順位を付け、時間配分を設計する
- 各議題の目的（共有/議論/意思決定）を明確にする
- 事前資料の準備と配布を管理する

### 論点整理

- 議題を意思決定に必要な論点に分解する
- 各論点に対する選択肢とトレードオフを整理する
- 不足している情報・データを特定する

### 意思決定フレームワーク

- 決定事項・保留事項・次のアクションを明確に記録する
- 意思決定の根拠と前提条件を文書化する
- フォローアップ項目と担当・期限を設定する

## 出力フォーマット

```text
## 経営会議アジェンダ（YYYY-MM-DD）

### 議題一覧
| # | 議題 | 目的 | 時間 | 担当 |
| - | ---- | ---- | ---- | ---- |

### 論点整理
#### 議題 1: [タイトル]

- 論点: ...
- 選択肢 A: ... → メリット / デメリット
- 選択肢 B: ... → メリット / デメリット
- 推奨: ...

### 決定事項
| # | 決定内容 | 担当 | 期限 |
| - | -------- | ---- | ---- |
```text
