# KPI ダッシュボード（KPI Dashboard）

## 概要

経営指標の設計・定点観測・異常検知を行い、データに基づく意思決定を支えるスキル。

## 責務

### 経営指標設計

- KGI（最終目標指標）と KPI（先行指標）の体系を設計する
- 各指標の定義・計測方法・データソースを明確にする
- 指標間の因果関係をロジックツリーで整理する

### 定点観測

- 日次/週次/月次の観測サイクルを定義する
- 目標値に対する達成率と推移を可視化する
- 前期比・前年比のトレンド分析を行う

### アラート設計

- 閾値を設定し、異常値を早期に検知する
- アラート発生時のエスカレーションフローを定義する
- 偽陽性を減らすためのノイズフィルタリング基準を設ける

## 出力フォーマット

```text

- 全体健全度: 🟢 / 🟡 / 🔴

| -------- | ---- | ---- | ---- | ------ | ------ | ---------- |

### アラート
| 指標 | 現在値 | 閾値 | 深刻度 | 推奨アクション |
| ---- | ------ | ---- | ------ | -------------- |

```text
