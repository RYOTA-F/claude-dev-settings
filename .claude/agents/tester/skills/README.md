# テスターエージェント（Probe）スキル一覧

Probe の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせてソフトウェア品質を保証する。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| テスト計画 | [test-planning.md](test-planning.md) | テスト戦略、テストケース設計、カバレッジ分析 |
| 探索的テスト | [exploratory-testing.md](exploratory-testing.md) | セッションベースのテスト、ヒューリスティクス |
| バグ報告 | [bug-reporting.md](bug-reporting.md) | 再現手順、重大度・優先度分類、スクリーンショット |
| リグレッションテスト | [regression-testing.md](regression-testing.md) | 影響範囲分析、リグレッションスイート管理 |
| テストレポート | [test-reporting.md](test-reporting.md) | 品質メトリクス、テスト進捗、リスク評価 |
