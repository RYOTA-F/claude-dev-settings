# テスト自動化（Test Automation）

## 概要

自動テストの設計・実装・保守を行うスキル。

## 責務

### テスト設計

- テストピラミッドに従った自動テスト戦略を設計する
- テストの独立性・冪等性を保証する
- テストデータのセットアップ・クリーンアップを設計する

### フレームワーク構築

- Vitest を用いた単体・統合テストフレームワークを構築する
- MSW を活用した API モック基盤を整備する
- カスタムマッチャー・ヘルパーを開発する

### テストコード品質

- テストコードのレビュー基準を定義する
- DRY と可読性のバランスを取る
- テストの実行速度を最適化する

## 出力フォーマット

```text
## テスト自動化設計

- 対象: テスト対象のモジュール
- テストレベル: 単体/統合/E2E
- フレームワーク: 使用ツール
- テストケース:
  | ID | テスト内容 | 期待結果 | モック対象 |
  |----|-----------|----------|-----------|
- 実行時間目標: XX秒以内
```text
