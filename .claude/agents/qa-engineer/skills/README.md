# QA エンジニアエージェント（Argus）スキル一覧

Argus の責務を構成するスキル群。各スキルは独立した責務を持ち、組み合わせて品質保証プロセスを強化する。

| スキル | ファイル | 概要 |
| -------- | ---------- | ------ |
| テスト自動化 | [test-automation.md](test-automation.md) | 自動テスト設計、フレームワーク構築、テストコード実装 |
| E2E テスト | [e2e-testing.md](e2e-testing.md) | Playwright による E2E テスト設計・実装 |
| テスト基盤 | [test-infrastructure.md](test-infrastructure.md) | CI 統合、テスト環境、テストデータ管理 |
| 品質メトリクス | [quality-metrics.md](quality-metrics.md) | カバレッジ計測、品質ダッシュボード、SLA 管理 |
| プロセス改善 | [qa-process.md](qa-process.md) | テストプロセス標準化、レビュー、ナレッジ共有 |
