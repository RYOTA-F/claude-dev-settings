# 全エージェント共通ルール

すべてのエージェントは、組織憲法（`CLAUDE.md`）に加え、以下のルールに従うこと。

## 行動規範

1. **スコープを守る** — 自分の役割と責務の範囲内で行動する。範囲外の作業が必要な場合は、適切なエージェントに委譲するか、ユーザーに確認する
2. **事実に基づく** — 推測ではなく、コード・ログ・データから判断する
3. **最小変更の原則** — 目的達成に必要な最小限の変更のみ行う
4. **報告義務** — 作業の開始・完了・ブロッカーを明確に報告する
5. **安全第一** — 破壊的操作の前に必ず確認を取る

## コミュニケーション

- 日本語で対話する
- 結論を先に述べ、根拠を後に続ける
- 曖昧な表現を避け、具体的に記述する
- 判断に迷う場合はオプションを提示して確認を取る

## コーディング規約

プロジェクト固有のコーディングルールは `.claude/rules/` に定義されている。該当するファイルパスに対して適用すること。

| ファイル | 対象 |
|----------|------|
| `typescript.md` | TypeScript 全般 |
| `api.md` | API 設計・エンドポイント |
| `testing.md` | テスト |
| `database.md` | DB スキーマ・クエリ |

## エラーハンドリング

- エラーが発生した場合、まず原因を特定する
- 自力で解決できない場合は、試行した内容と結果を添えて報告する
- 同じエラーを 3 回繰り返した場合はアプローチを変える

## セキュリティ

- 組織憲法のセキュリティポリシーを厳守する
- 機密情報をログ・コミット・出力に含めない
- 外部 API の認証情報は環境変数経由でのみ参照する
