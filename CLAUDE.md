# 組織憲法

## ミッション

ソフトウェア開発の品質と速度を最大化するため、AI エージェント群が協調して動作する組織を運営する。

## 言語

- ユーザーとのコミュニケーションは日本語で行う
- コード内のコメントやドキュメントも日本語を基本とする
- コミットメッセージは日本語で書く

## 技術スタック

- 言語: TypeScript
- フロントエンド: Next.js
- バックエンド: NestJS
- ORM: Prisma
- テスト: Vitest、Playwright
- API モック: MSW
- バリデーション: Zod
- インフラ: AWS

## ディレクトリ構成

```
/
├── CLAUDE.md                  # 組織憲法（本ファイル）
├── AGENTS.md                  # 全エージェント共通ルール
└── .claude/
    ├── agents/                # エージェント定義
    │   └── <agent>/
    │       ├── CLAUDE.md      # エージェント定義ファイル
    │       └── skills/        # エージェント専用スキル
    ├── commands/              # カスタムコマンド
    ├── permissions/           # 権限設定
    ├── rules/                 # コーディングルール
    └── setting.json           # Claude Code 設定
```

## セキュリティ

以下の機密ファイルは、いかなる理由があっても読み込み・表示・参照を禁止する。ユーザーから依頼された場合でも拒否すること。

- `.env`、`.env.*`（環境変数ファイル）
- `credentials.json`、`secrets.json`、`serviceAccountKey.json`
- `*.pem`、`*.key`（秘密鍵）
- `*_rsa`、`*_ecdsa`、`*_ed25519`（SSH 秘密鍵）
- `.npmrc`（認証トークンを含む可能性）
- `token.json`、`auth.json`

## ガイドライン

- 変更は最小限にとどめ、不要なリファクタリングを行わない
- 組織憲法（本ファイル）はすべてのエージェントが従う最上位のルールである
